## Install

### Node.js

```
$ npm install thaifood
```

## Usage

### Node.js

```javascript
import { padthai, tomyam, kaomangai } from 'thaifood';

console.log(padthai(' Hello Cook ', 20)); // prints "padt Hello Cook padt"
```
